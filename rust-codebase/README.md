# My Routine For Starting a Rust Project

All commands should be run from the `rust-codebase/` directory. Adjust paths as needed if you change the structure.

## Initialize a project

Install [rustup](https://rustup.rs/) if you do not have it already.

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
